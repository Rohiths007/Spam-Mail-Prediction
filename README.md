# 📧 Email Spam Classification – Learning Project

This repository contains my learning implementation of an Email Spam Classification model using multiple machine learning algorithms.

The main goal of this project was to understand how different classification algorithms work, compare their performance, and gain hands on experience with text classification.and the Dataset Source is taken in Kaggle.

## 📚 What I Learned

- Text preprocessing
- Data cleaning
- Train-Test Split
- Feature extraction using TF-IDF
- Model training and evaluation
- Comparing multiple machine learning algorithms

## 🤖 Algorithms Explored

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Support Vector Machine (SVM)
- XGBoost
- LightGBM
- CatBoost

## 📊 Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- LightGBM
- CatBoost
- Jupyter Notebook

## 🎯 Purpose

This is a **learning project** created to strengthen my understanding of machine learning classification algorithms and text classification. It is not intended as a production-ready spam detection system.
